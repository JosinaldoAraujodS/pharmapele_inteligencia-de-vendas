🛒 Venda Casada

Identifica pares de produtos vendidos na mesma nota. Cada combo aparece uma única vez (A+B = B+A), agrupando cadastros com o mesmo nome. A confiança indica, em cada sentido, com que frequência quem leva um produto também leva o outro — ideal para combos, kits e script de balcão.

Combinações encontradas ≥40% forte · ≥20% moderada · <20% fraca
🔔 Alertas de Recompra

Cada alerta é calculado por cliente + produto: o sistema analisa quanto tempo esse cliente leva para recomprar aquele produto específico (mínimo 2 compras no histórico). A data estimada = última compra + intervalo médio entre as compras anteriores.
